Tipos de Bucles

Bucle for ‚ñ∂

Se utiliza cuando sabemos cu√°ntas veces queremos repetir una acci√≥n o cuando queremos iterar sobre una secuencia de elementos.

¬øQu√© hace este c√≥digo?
Crea una lista de frutas y usa un bucle for para recorrer cada elemento de la lista. En cada iteraci√≥n, imprime el nombre de la fruta.
¬øCu√°ndo usarlo?
Cuando necesitas procesar todos los elementos de una colecci√≥n (lista, tupla, string, etc.).
Bucle while ‚ñ∂

Se utiliza cuando queremos repetir una acci√≥n mientras una condici√≥n sea verdadera.

while contador < 5:
    print(contador)
    contador += 1
¬øQu√© hace este c√≥digo?
Inicializa una variable contador en 0 y la incrementa en cada iteraci√≥n mientras sea menor que 5. Imprime el valor de contador en cada paso.
¬øCu√°ndo usarlo?
Cuando no sabes cu√°ntas veces se repetir√° la acci√≥n y depende de una condici√≥n l√≥gica.
Bucles Anidados ‚ñ∂

Son √∫tiles para trabajar con estructuras de datos multidimensionales o para realizar operaciones que requieren m√∫ltiples iteraciones.

# Tabla de multiplicaci√≥n

        print(f"{i} x {j} = {i * j}")
¬øQu√© hace este c√≥digo?
Usa dos bucles for anidados para imprimir una tabla de multiplicar del 1 al 3. El bucle externo controla la fila (i) y el interno la columna (j).
¬øCu√°ndo usarlo?
Cuando necesitas recorrer estructuras de datos en dos dimensiones (matrices, tablas) o realizar combinaciones de elementos.
Control de Bucles (break, continue, else) ‚ñ∂

break: Termina el bucle inmediatamente.
continue: Salta a la siguiente iteraci√≥n del bucle.
else: Se ejecuta cuando el bucle termina normalmente (sin break).

# Ejemplo de break y continue
for numero in range(10):
    if numero == 5:
        break
    if numero % 2 == 0:
        continue
    print(numero)

# Ejemplo de else
for numero in range(5):
    print(numero)
else:
    print("Bucle completado")
¬øQu√© hace este c√≥digo?
El primer bloque imprime solo los n√∫meros impares menores que 5 y se detiene cuando numero es igual a 5. El segundo bloque imprime los n√∫meros del 0 al 4 y luego muestra un mensaje usando else al finalizar el bucle.
¬øCu√°ndo usarlo?
Cuando necesitas controlar el flujo del bucle, saltar iteraciones o ejecutar acciones al finalizar.
